Web Vulnerability Scanner — комплексный инструмент для аудита безопасности сайтов. Проверяет SSL/TLS, уязвимые библиотеки (jQuery, React), открытые файлы (.env, .git), XSS-уязвимости, DoS-защиту, открытые порты, SEO (title, meta, H1/H2), производительность и HTML-валидацию. Каждая проблема сопровождается подробной инструкцией по исправлению с готовым кодом для Apache/Nginx и прямыми ссылками. Темный интерфейс с прогресс-баром, экспорт в PDF/CSV. Только для авторизованного тестирования.
