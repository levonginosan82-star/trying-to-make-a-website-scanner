# website-scanner
Web Vulnerability Scanner —  инструмент аудита безопасности сайтов. Проверяет SSL/TLS сертификаты, уязвимые библиотеки (jQuery, React), открытые файлы, XSS, DoS-защиту, порты, SEO и HTML. Каждая проблема с подробной инструкцией по исправлению и кодом. Темный интерфейс с экспортом PDF/CSV. Только для авторизованного тестирования.
